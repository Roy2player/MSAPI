Example MSAPI Application View

Integration Instructions

To integrate your app with MSAPI Manager, include the iframe-bridge.js script:

<script src="/iframe-bridge.js"></script>

This script will:

  • Receive initialization messages from the parent manager window
  • Forward click events to update the z-index of your app view

What the script does:

  1. Listens for "init" messages containing the parent origin and unique ID
  2. Stores these values for communication with the parent window
  3. Forwards click events to the parent to bring your app view to front
